### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Danyaal-02/reactjs-admin-dashboard.git
   ```
2. Navigate to the project directory
   ```sh
   cd reactjs-admin-dashboard
   ```
3. Install NPM packages
   ```sh
   npm install
   ```

### Usage

To run the development server:

```sh
npm run dev
```
